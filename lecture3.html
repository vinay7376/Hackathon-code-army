Lecture 3: Lists & Tables

This lecture teaches you how to create ordered, unordered lists and tables in HTML.

🎥 Lecture Video

📝 Lecture Notes

  • Ordered List: <ol> with <li>
  • Unordered List: <ul> with <li>
  • Tables: <table>, <tr>, <th>, <td>

📄 Download Notes (PDF) | 🌐 View on Notion

💻 Example Code

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>25</td></tr>
</table>

✅ Key Takeaways

  • Lists help structure items.
  • Tables organize tabular data.
