Lecture 2: HTML Basics (Headings, Links, Images)

Learn how to use headings, paragraphs, links, and images in HTML.

🎥 Lecture Video

📝 Lecture Notes

  • Heading tags: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="url">Text</a>
  • Images: <img src="path" alt="description">

📄 Download Notes (PDF) | 🌐 View on Notion

💻 Example Code

<h1>Hello</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Sample Image">

✅ Key Takeaways

  • Headings structure your content.
  • Links allow navigation.
  • Images make your page visual.
