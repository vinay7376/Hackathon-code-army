Lecture 5: Forms

This lecture covers how to create forms for user input in HTML including text fields, radio buttons, checkboxes, and submit buttons.

🎥 Lecture Video

📝 Lecture Notes

  • <form> element for creating forms
  • Input types: text, password, email, radio, checkbox
  • <textarea> for multiline text
  • <select> and <option> for dropdowns
  • <button> or <input type="submit"> to submit

📄 Download Notes (PDF) | 🌐 View on Notion

💻 Example Code

<form>
  <label>Name:</label>
  <input type="text"><br>
  <label>Gender:</label>
  <input type="radio" name="gender"> Male
  <input type="radio" name="gender"> Female <br>
  <input type="submit" value="Submit">
</form>

✅ Key Takeaways

  • Forms collect user input.
  • Input types are important for correct data.
  • Forms are the backbone of interactive websites.
